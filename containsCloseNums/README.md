Dado un array de enteros, determinar si existen dos indeces distintos.

**EJEMPLO**

Para nums[0, 1, 2, 3, 5, 2] y k = 2, containsCloseNums(nums, k) = false.

Para nums[0, 1, 2, 3, 5, 2] y k = 3, containsCloseNums(nums, k) = true.

Para nums[0, 1, 2, 3, 5, 2] y k = 2, containsCloseNums(nums, k) = false.

Para nums[0, 1, 2, 3, 5, 2] y k = 2, containsCloseNums(nums, k) = false.
